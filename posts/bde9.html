Windows和IDEA的常用快捷键

发表于 2021-08-17 更新于 2021-09-17 分类于 工具 阅读次数： 评论次数：
本文字数： 984 阅读时长 ≈ 1 分钟

博主作为一个Java后端攻城狮，开发过程中不免要使用许多的快捷键。合理使用快捷键能极大提高工作效率，那么这么多的快捷键又该记住哪些呢，把所有快捷键都记下显然是不太现实的。因此我根据平时的使用习惯，总结了自己在Windows和IDEA中常用的快捷键。

Windows快捷键

快捷键 功能描述
win 打开开始菜单
win + d 缩小所有窗口
win + a 打开右侧通知栏(可连wifi、蓝牙等)
win + e 打开我的电脑
win + r cmd进入命令行，regedit进入注册表
win + q 打开搜索栏
win + l 快速锁屏
win + i 打开windows设置
win + x + u + s 使电脑进入睡眠状态
win + p 多屏管理
win + . 输入表情符号
alt + tab 切换窗口
ctrl + shift + esc 打开任务管理器
ctrl + tab 切换chrome网页窗口
ctrl + j 进入下载内容
ctrl + d 收藏网页
F5 chrome刷新网页
F12 chrome进入开发者工具
ctrl + r 开发者工具下刷新页面请求
F2 重命名
ctrl + alt + a 【个人习惯】截屏
F3 【个人习惯】贴图
鼠标左侧按键 【个人习惯】网页前进后退
ctrl + 选中文字 【个人习惯】沙拉查词翻译
ctrl + / 【个人习惯】语雀插入代码块
ctrl + ` 打开Ditto，双击即复制

IDEA快捷键

快捷键 功能描述
选中项目 + → 展开项目
ctrl + z 撤销
ctrl + shift + z 反撤销
ctrl + y 删除光标所在行
ctrl + d 复制当前行
鼠标快速点三下 选中当前行
ctrl + alt + t 快速try…catch/if
ctrl + / 单行注释
ctrl + shift + / 多行注释
alt + enter 导包、修正代码
alt + insert pojo类生成方法
ctrl + tab 切换代码窗口(按上下键选择文件)
ctrl + f 当前文件搜索关键字
ctrl + shift + f 全局搜索关键字
ctrl + r 当前文件替换关键字
ctrl + shift + r 全局替换关键字
ctrl + n 通过类名查找文件
ctrl + shift + n 通过文件名定位
ctrl + alt + l 调整代码格式
ctrl + F12 查看类结构(字段和方法)
ctrl + h 查看类继承关系
shift + F6 重命名文件
shift + F9 Debug启动项目
ctrl + t 【个人习惯】CodeIf命名插件
F8 代码向下执行一行
F7 进入到某个方法内部，F8向下执行可跳出
F9 执行到下一个断点或执行到底
alt + F8 断点调试时查看某行代码数据
IDEA Windows 快捷键
搭建自己的个人博客网站(Github+Hexo+Next)
